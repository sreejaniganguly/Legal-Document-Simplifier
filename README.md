Legal-document-Simplifier is a lightweight web application that converts complex legal language into clear, human-readable text using AI.

WHAT IT DOES :
1. It simplifies difficult legal texts into plain English as well as in Hindi and Bengali
2. Shows an overall risk score and summary
3. Fast and user friendly interface
4. Works with pdf input
5. Clean and minimal UI

TECH STACK USED :
1. Frontend- HTML, CSS,Javascript
2. Backend- Node.js
3. Version Control- Git & GitHub

   SETUP
   
 Run locally:
 git clone 
 https://github.com/sreejaniganguly/Legal-Document-Simplifier.git
cd Legal-Document-Simplifier
npm install
node server.js

Open:
http://localhost:3000

API Keys :
Uses AI APIs for text simplification
Store keys in .env file
Never expose keys publicly
